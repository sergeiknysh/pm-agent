## Telegram уведомления

Настроил скрипт `scripts/notify-telegram.sh` для отправки уведомлений в Telegram когда заканчиваю большие задачи.

**ВАЖНО:** В конце каждой большой задачи вызывать:
```bash
./scripts/notify-telegram.sh "✅ [описание что сделано]"
```
